Lab 3 - P

  • Due 1 Sep by 23:59
  • Points 0
  • Submitting a file upload

This lab allows you to practice subnetting again and investigate how ARP works and how the CAM table of a switch is populated. You will use Packet Tracer (PT) to build the network and observe the tables being populated.
